Documentação do Sistema

Introdução

Este sistema foi desenvolvido para auxiliar na análise de projetos técnicos de chapas metálicas. Abaixo estão as instruções de uso.

Como usar

  1. Acesse o painel com seu login institucional.
  2. Envie o desenho técnico no formato suportado (.png, .jpg, .pdf).
  3. Preencha os campos de matéria-prima e processos, caso deseje.
  4. A IA irá gerar automaticamente observações com base no desenho enviado.

Dicas

  • Use desenhos com boa resolução.
  • Evite enviar arquivos escaneados com sombras ou ruídos.
  • Preencha as observações manuais para maior controle do projeto.
