Documentação do Sistema

Introdução

Este sistema foi desenvolvido para auxiliar na análise de projetos mecânicos. Abaixo estão as instruções de uso. O sistema utiliza inteligência artificial para gerar automaticamente a matéria prima e os processos de fabricação necessários para chegar ao produto final com base no desenho enviado.

Como usar

  1. Acesse o painel com seu login institucional.
  2. Envie o desenho técnico no formato suportado (.png, .jpg).
  3. selecione a quantidde de peças que serão fabricadas
  4. Preencha o campo de observações adicionais, caso deseje.
  5. Aguarde o processamento da IA.
  6. Após o processamento, você receberá todas as informações para fabricação do projeto
  7. Você pode verificar todas as suas análises na aba Projetos.

Dicas

  • Use desenhos com boa resolução.
  • Evite enviar arquivos escaneados com sombras ou ruídos.
  • Preencha as observações manuais para maior controle do projeto.
